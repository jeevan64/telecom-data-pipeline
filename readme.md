# Telecom Data Pipeline Project

## Overview
This project simulates a telecom data engineering pipeline using:

- Python
- Pandas
- PySpark
- FastAPI
- Oracle SQL
- Git

## Features
- Data cleaning using Python
- Data transformation using PySpark
- API layer using FastAPI
- Oracle database integration
- Pipeline orchestration script
- KPI generation
